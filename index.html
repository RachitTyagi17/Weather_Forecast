Weather App

Welcome to the Weather App. This Python-based application fetches real-time weather details for any city using the OpenWeatherMap API.

How to Use the Application

  1. Clone the repository: 
    git clone https://github.com/RachitTyagi17/Weather_App.git
  2. Navigate to the project directory: 
    cd Weather_App
  3. Install the required dependencies: 
    pip install requests
  4. Replace `YOUR_API_KEY` in `weather_app.py` with your OpenWeatherMap API key.
  5. Run the application: 
    python weather_app.py
